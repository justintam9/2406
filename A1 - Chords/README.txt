This program transposes chords of a given song up or down.

Tested On:
Node v8.12.0
Windows 10 Pro

Install:
None

Launch:
node server.js

Testing:
Visit http://localhost:3000/assignment1.html
Type any of the hardcoded songs (Sister Golden Hair, Peaceful Easy Feeling, Brown Eyed Girl) into the text box and click Submit.
