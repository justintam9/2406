This program uses the food2fork API to gather recipes with a given keyword and displays all recipes using the submitted ingredient.

Tested On:
Node v8.12.0
Windows 10 Pro

To install npm modules execute:
>npm install

This will install the modules listed as dependencies in the package.json file.

To run either execute in a command prompt
>npm start
or 
>node server.js

Testing:
Use Chrome browser to visit:
Visit any of the following:
